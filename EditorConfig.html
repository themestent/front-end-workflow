EditorConfig

Including EditorConfig file to maintain a consistent coding styles between different editors and IDEs. Contribution is growing in great numbers and we already have plugin for popular IDEs and editors.

If you never heard of EditorConfig before, please read this article Maintain Consistent Coding Styles Between Different Editors and IDEs written by Hong-Xu one of the co-creator of EditorConfig.

Here are the settings used in Front-End Workflow project .editorconfig file:

# editorconfig.org

root = true

[*]
indent_style = space
indent_size = 2
end_of_line = lf
charset = utf-8
trim_trailing_whitespace = true
insert_final_newline = true

[*.py]
indent_style = space
indent_size = 4
