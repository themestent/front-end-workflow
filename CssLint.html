CSSLint

We can automatically point out problems with our CSS code with CSSLint which does syntax checking with a set of defined pluggable rules. CssLint tries to find problematic patterns, possible errors, compatibility problems across browsers or performance inefficiencies in your written codes.

Front-End-Workflow created a task called lint to automate this process using grunt-contrib-csslint package. Rules are defined in .csslintrc file.
